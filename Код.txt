
Знайти множники числа 
print_factors <- function(x) {
print(paste("The factors of",x,"are:"))
for(i in 1:x) {
if((x %% i) == 0) {
print(i)
}
}
}

У цій програмі ми беремо число та відображаємо його множники за допомогою функції print_factors().
У функції ми використовуємо forцикл для ітерації від 1 до цього числа і друкуємо його тільки в тому випадку, 
якщо воно ідеально ділить наше число. Тут print_factors() - функція, що визначається користувачем.
